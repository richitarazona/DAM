CSS3

CSS3 es una versión posterior a CSS2 que trae más propiedades, que ayudan a mejorar el diseño Web, ya que se pueden hacer más cosas con menos esfuerzo; ahora bien, no todos los navegadores lo soportan, especialmente los antiguos.

Can I Use

Mientras CSS3 se convierte en un estándar, los distintos navegadores pueden incluir o no las mejoras propuestas, o incluso añadir las suyas propias. La web Can I use...incluye un listado completo de funcionalidades soportadas por los distintos navegadores.

Prefijos de navegadores

Los prefijos de navegadores nos proporcionan una rápida y fácil implementación de propiedades estándar y sintaxis alternativas. Estos prefijos permiten a los navegadores implementar propiedades CSS3 que de otra forma no funcionan de forma estándar, por el momento. Por este motivo, los prefijos cobran un papel importante en numerosas características de CSS3. Por ejemplo: propiedades de animación, bordes con esquinas redondeadas, sombras en cajas, etc.

Prefijos de cada navegador:

  • WebKit: -webkit /* Google chrome*/
  • Firefox: -moz /*Mozilla*/
  • Opera: -o /*Opera*/
  • Internet Explorer: -ms */Internet explorer*/

Para añadir los prefijos con Sublime Text, preceder con un guión(-) a la propiedad y pulsar Tabulador. Ejemplo: -border-radius(tab), aparecerán todos los prefijos. Si únicamente quiero limitar a unos pocos: guión+inicialesnavegador+propiedad+tab. Ejemplo: 

     
        -border-radius+tabulador, el resultado es:
        -webkit-border-radius: ;
        -moz-border-radius: ;
        -ms-border-radius: ;
        -o-border-radius: ;
        border-radius: ;
        -wm-border-radius+tabulador, el resultado es:
        -webkit-border-radius: ;
        -moz-border-radius: ;
        border-radius: ;

Selectores

La versión actual de CSS 3 incluye todos los selectores de CSS 2.1 y añade otras decenas de selectores, pseudo-clases y pseudo-elementos. La lista provisional de novedades y su explicación detallada se puede encontrar en el módulo de selectores de CSS 3.

Aquí puedes consultar la una breve explicación de Selectores

Sectores de atributo

Mediante los selectores de atributos podemos seleccionar elementos que disponen de ese atributo y cuyo valor comienza ^, contiene *, o termina $, por la cadena de texto indicada

Estos son algunos ejemplos

              
     
        /* Selecciona todos los enlaces que apuntan a una dirección de correo
        electrónico */
        a[href^="mailto:"] { ... }
     
        /* Selecciona todos los enlaces que apuntan a una página HTML */
        a[href$=".html"] { ... }
     
        /* Selecciona todos los títulos h1 cuyo atributo title contenga la palabra
        "capítulo" */
        h1[title*="capítulo"] { ... }

Selector general de hermanos

Otro de los nuevos selectores de CSS 3 es el "selector general de elementos hermanos", que generaliza el selector adyacente de CSS 2.1. Su sintaxis es elemento1 ~ elemento2 y selecciona el elemento2 que es hermano de elemento1 y se encuentra detrás en el código HTML. En el selector adyacente la condición adicional era que los dos elementos debían estar uno detrás de otro en el código HTML, mientras que ahora la única condición es que uno esté detrás de otro 

     
        h1 + h2 { ... }  /* selector adyacente */
        h1 ~ h2 { ... }  /* selector general de hermanos */

Pseudoelementos y Pseudoclases

Los pseudo-elementos de CSS 2.1 se mantienen en CSS 3, pero cambia su sintaxis y ahora se utilizan :: en vez de : delante del nombre de cada pseudo-elemento.

  • ::first-line
  • ::first-letter
  • ::before
  • ::after
  • ::selection

Tiene 12 nuevas pseudoclases:

  • elemento:nth-child(numero)
  • elemento:nth-child(odd)
  • elemento:nth-child(even)
  • elemento:nth-last-child(numero)
  • elemento:empty
  • elemento:first-child
  • elemento:last-child
  • elemento:nth-of-type(numero)
  • elemento:nth-last-of-type(numero)
