Variables para dos temas que crearemos. Para cada tema establece el color base, el color de fa fuente, color de fondo y color de borde, utiliza funciones de less para actuar siempre sobre el color base.

Mixins para establecer un borde (con todos sus posibles parámetros), para crear tema(como parametros, el color de la fuente, fondo, y bode con todos sus parámetros, mixin anterior), crear una caja(ancho como parámetro), darle un ancho máximo a una caja(como parámetro dicho ancho) y márgenes equitativos a derecha e izquierda y mixin para flotar a un lado(lado como parámetro)

Utiliza un archivo para variables y mixins otro para lo demás utilizando comentarios y compila todo en styles.less

Para crear el efecto de cortar lo que no cabe y poner puntos suspensivos utiliza las propiedades: 


    white-space: nowrap;
    overflow: hidden;
    // ellipsis aparecen los tres puentos suspensivos al ocultar
    text-overflow: ellipsis;
